# API Endpoints

The generation API is defined in `src/routes/generate.py` and provides two endpoints for triggering and monitoring idea generation.

## Generate Ideas

Enqueue a Celery task to generate synthetic project ideas.

```
POST /api/generate-ideas
```

**Request Body:**

```json
{
  "num_ideas": 5,
  "seed_tags": ["nutrition", "ios"],
  "creativity": 0.85
}
```

| Field | Type | Default | Description |
|---|---|---|---|
| `num_ideas` | `int` | `5` | Number of ideas to generate (1--50) |
| `seed_tags` | `string[] \| null` | `null` | Tags to guide generation |
| `creativity` | `float` | `0.85` | LLM temperature (0.1--1.5) |

**Response:** `200 OK`

```json
{
  "task_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "num_requested": 5,
  "seed_tags": ["nutrition", "ios"]
}
```

The `task_id` is used to poll for progress and results.

## Poll Task Status

Check the status of a running or completed generation task.

```
GET /api/generate-ideas/{task_id}/status
```

**Path Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `task_id` | `string` | Celery task ID from the generate endpoint |

### Response States

**PENDING** -- Task is queued but not yet started:

```json
{
  "task_id": "a1b2c3d4-...",
  "state": "PENDING"
}
```

**STARTED** -- Task is running with progress metadata:

```json
{
  "task_id": "a1b2c3d4-...",
  "state": "STARTED",
  "stage": "generating",
  "progress": 3,
  "total": 5
}
```

The `stage` field indicates the current phase: `building_pipeline`, `generating`, or `saving`.

**SUCCESS** -- Task completed with results:

```json
{
  "task_id": "a1b2c3d4-...",
  "state": "SUCCESS",
  "result": {
    "status": "success",
    "num_requested": 5,
    "num_generated": 5,
    "num_saved": 5,
    "project_ids": [101, 102, 103, 104, 105],
    "errors": [],
    "seed_tags": ["nutrition", "ios"]
  }
}
```

**FAILURE** -- Task failed:

```json
{
  "task_id": "a1b2c3d4-...",
  "state": "FAILURE",
  "error": "Error generating dataset: LLM endpoint unreachable"
}
```

## Filtering Generated Projects

Generated projects can be retrieved using the existing projects endpoint with the `source` filter:

```
GET /api/waywo-projects?source=nemo_data_designer
```

This returns only AI-generated projects. See [Projects API](/api/projects) for the full filter options.
