# Projects

Projects are structured records extracted from HN comments by the AI workflow. Each project has a title, description, hashtags, scores, and optional bookmark status. The projects endpoint supports the richest set of filters in the API.

## List Projects

Retrieve a paginated, filterable list of projects.

```
GET /api/waywo-projects
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|---|---|---|---|
| `limit` | `int` | `50` | Maximum results to return |
| `offset` | `int` | `0` | Number of results to skip |
| `comment_id` | `int \| null` | `null` | Filter by source comment ID |
| `tags` | `string \| null` | `null` | Comma-separated hashtags (e.g. `cli,devtools`) |
| `min_idea_score` | `int \| null` | `null` | Minimum idea score (1-10) |
| `max_idea_score` | `int \| null` | `null` | Maximum idea score (1-10) |
| `min_complexity_score` | `int \| null` | `null` | Minimum complexity score (1-10) |
| `max_complexity_score` | `int \| null` | `null` | Maximum complexity score (1-10) |
| `is_valid` | `bool \| null` | `null` | Filter by validation status |
| `bookmarked` | `bool \| null` | `null` | Filter by bookmark status |
| `date_from` | `datetime \| null` | `null` | Projects created on or after this date (ISO 8601) |
| `date_to` | `datetime \| null` | `null` | Projects created on or before this date (ISO 8601) |
| `sort` | `string \| null` | `null` | Sort order: `random` for random, default is newest first |
| `source` | `string \| null` | `null` | Filter by source: `hn` or `nemo_data_designer` |

**Example Request:**

```
GET /api/waywo-projects?tags=cli,rust&min_idea_score=7&is_valid=true&limit=10
```

## Get Single Project

Retrieve full project details including the source comment and parent post.

```
GET /api/waywo-projects/{project_id}
```

**Path Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `project_id` | `int` | Project ID |

**Response:** `200 OK`

## Delete Project

Permanently delete a project record.

```
DELETE /api/waywo-projects/{project_id}
```

**Path Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `project_id` | `int` | Project ID to delete |

**Response:** `200 OK`

```json
{
  "status": "deleted",
  "project_id": 42,
  "message": "Project 42 has been deleted"
}
```

**Error:** `404` if the project ID does not exist.

## Toggle Bookmark

Toggle the bookmark status of a project. If currently bookmarked, it becomes unbookmarked, and vice versa.

```
POST /api/waywo-projects/{project_id}/bookmark
```

**Path Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `project_id` | `int` | Project ID to bookmark/unbookmark |

**Response:** `200 OK`

```json
{
  "is_bookmarked": true,
  "project_id": 42
}
```

**Error:** `404` if the project ID does not exist.

## List Hashtags

Retrieve all unique hashtags used across projects. Useful for building tag autocomplete or filter UIs.

```
GET /api/waywo-projects/hashtags
```

**Parameters:** None

**Response:** `200 OK`

```json
{
  "hashtags": ["ai", "cli", "devtools", "fintech", "ml", "open-source", "rust", "saas", "web"],
  "total": 9
}
```
