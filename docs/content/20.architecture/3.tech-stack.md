## AI / ML Models

Waywo uses three NVIDIA Nemotron models, each served on separate GPU infrastructure and accessed over HTTP.

| Model | Parameter Count | Role | Dimension / Output |
|---|---|---|---|
| **NVIDIA-Nemotron-3-Nano-30B-A3B-BF16** | 30B (3B active MoE) | LLM -- project extraction, validation, metadata, scoring, chatbot response | Text generation |
| **llama-embed-nemotron-8b** | 8B | Embedding -- semantic representation of projects and queries | 4096-dim FLOAT32 vector |
| **llama-nemotron-rerank-1b-v2** | 1B | Reranking -- cross-encoder relevance scoring for search results | Relevance score per document |

::tip
The LLM is accessed via an OpenAI-compatible API (`/v1/completions`), so any vLLM or TGI-served model can be swapped in by changing `LLM_BASE_URL` and `LLM_MODEL_NAME`. The embedding and rerank services expose simple `/embed` and `/rerank` REST endpoints.
::

## Python Version and Build

The project uses **Python 3.12+** with `setuptools` as the build backend. Dependencies are declared in `pyproject.toml` with optional groups:

- **Default** -- all runtime dependencies
- **`dev`** -- testing, linting, and formatting tools
- **`notebook`** -- Jupyter Lab for interactive exploration
