# Code Style

## Python -- Black + isort

The Python codebase is formatted with [Black](https://black.readthedocs.io/) and import-sorted with [isort](https://pycqa.github.io/isort/) (configured to be Black-compatible).

### Configuration

Both tools are configured in `pyproject.toml`:

```toml
[tool.black]
line-length = 88
target-version = ['py312']

[tool.isort]
profile = "black"
line_length = 88
```

### Formatting Commands

```bash
# Format all Python code (via Docker)
make black

# Check formatting without modifying (via Docker, useful in CI)
make check
```

To run directly without Docker:

```bash
# Format
black src/

# Check only
black --check src/

# Sort imports
isort src/
```

### Additional Tools

The `[dev]` dependency group also installs:

| Tool | Purpose |
|------|---------|
| `black` | Code formatting |
| `isort` | Import sorting |
| `flake8` | Linting (available but no custom rules enforced) |
| `mypy` | Static type checking (available but not enforced in CI) |

## Frontend -- ESLint

The Nuxt frontend uses ESLint with the `@nuxt/eslint` preset. The configuration is minimal, relying on Nuxt defaults:

```js
// eslint.config.mjs
import withNuxt from './.nuxt/eslint.config.mjs'

export default withNuxt(
  // Your custom configs here
)
```

### Running ESLint

```bash
cd frontend
npx eslint .
```

No additional linting rules are configured beyond the `@nuxt/eslint` defaults.

## Conventions

### Python

- **Line length**: 88 characters (Black default)
- **Quotes**: Double quotes (Black default)
- **Imports**: Sorted with isort in Black-compatible mode
- **Type hints**: Used throughout, especially for Pydantic models and FastAPI signatures
- **Docstrings**: Present on all public API endpoints, workflow steps, and event classes

### Frontend

- **TypeScript**: All `<script setup>` blocks use `lang="ts"`
- **Component naming**: shadcn components in `components/ui/`, custom components in `components/waywo/`
- **Styling**: Tailwind CSS utility classes, no custom CSS files
- **Formatting**: Handled by ESLint auto-fix
